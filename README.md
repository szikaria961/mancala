# Project Title

Mancala

## Description

Basic Mancala App with react

## Development

### Installation

```
git clone https://github.com/szikaria961/mancala.git
cd mancala
npm install
```

### Client

Create a new terminal tab/session and do the following:

```
cd mancala
npm run dev
```

View development app at `http://localhost:8080`
